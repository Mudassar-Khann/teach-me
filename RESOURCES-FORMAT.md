# RESOURCES.md Format

`RESOURCES.md` is the curated set of trusted sources for this topic. Knowledge for explainers should be drawn from here, not from parametric guesses. Wisdom comes from the communities listed here.

## Structure

```md
# {Topic} Resources

## Knowledge

- [Book: _Introduction to Algorithms_ — Cormen, Leiserson, Rivest, Stein](https://mitpress.mit.edu/9780262046305/introduction-to-algorithms/)
  Foundational text on algorithm design and analysis. Use for: Big O notation, data structure implementation details.
- [Documentation: _Python 3 Official Docs_](https://docs.python.org/3/)
  The canonical source of truth for standard library implementations. Use for: syntax verification, built-in functions.

## Wisdom (Communities)

- [r/learnprogramming](https://www.reddit.com/r/learnprogramming/)
  High-signal subreddit for beginners. Use for: code review, debugging logic flaws, career advice.
- [Stack Overflow: Data Structures Tag](https://stackoverflow.com/questions/tagged/data-structures)
  Use for: highly specific implementation bugs and edge-case behaviors.
